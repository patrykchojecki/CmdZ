# Security Policy

## Supported versions

Security updates are provided for the latest released version of CmdZ.

| Version | Supported |
| --- | --- |
| 1.x | Yes |
| Earlier versions | No |

## Report a vulnerability

Please do not disclose suspected vulnerabilities in a public issue.

[Report a vulnerability privately through GitHub](https://github.com/patrykchojecki/CmdZ/security/advisories/new). Include:

- a description of the issue and its potential impact;
- the CmdZ and Chrome versions tested;
- clear reproduction steps or a proof of concept; and
- any suggested mitigation, if available.

The maintainer will acknowledge the report, investigate it, and coordinate a fix and disclosure as appropriate. CmdZ is a small volunteer-maintained project, so response times may vary.

## Security scope

CmdZ has no server, account system, analytics, or network access. Relevant reports generally concern the extension's source code, packaged release, permissions, or Chrome Web Store distribution.
